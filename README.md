# Dog Universe

[![CI](https://github.com/Archsider/dog-universe/actions/workflows/ci.yml/badge.svg)](https://github.com/Archsider/dog-universe/actions/workflows/ci.yml)

Application web de gestion de pension pour animaux (chiens et chats) — Maroc.

Stack : Next.js 15 · React 19 · Prisma · PostgreSQL (Supabase) · NextAuth · next-intl (fr/en).

## Démarrage

```bash
npm install
npx prisma generate
npm run dev
```

Voir `CLAUDE.md` pour la documentation complète du projet (architecture, conventions, historique).
